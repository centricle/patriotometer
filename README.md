# patriotometer

Political identity quiz with a satirical lens mechanic. 15 questions, 8 archetypes, 8 lenses, 64 vignettes, radar chart visualization, and Canvas meme download.

## Stack

- Astro 5 + TypeScript + Tailwind 4
- Static site, Netlify deploy
- No backend, no auth, no database

## Development

```bash
npm run dev      # Dev server on port 7029
npm run build    # Production build
```
